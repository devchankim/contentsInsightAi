# InsightWorkspace MVP - 비개발자용 기능 명세

> 이 문서는 PM/기획자가 기능을 확장하거나 요구사항을 추가할 때 참고하는 문서입니다.

## 1. 프로젝트 개요

**목표**: 개인 유튜버가 레퍼런스 영상 구조와 자신의 관점을 섞어, 촬영 가능한 2-Column 대본을 빠르게 만든다.

**핵심 성공 지표**: 제작시간 단축 (기획→초안: 30분 → 10분)

## 2. 주요 화면 및 기능

### 화면 1) 프로젝트 리스트 (홈)

**목적**: 내가 만든 대본 프로젝트를 관리

**주요 요소**
- 프로젝트 카드 (제목, 주제, 업데이트 시간, 생성 횟수)
- "새 프로젝트" 버튼
- 삭제 버튼 (카드 우측 상단)

**유저 액션**
1. "새 프로젝트" 클릭 → 자동 생성 → 워크스페이스로 이동
2. 프로젝트 카드 클릭 → 해당 프로젝트 워크스페이스 열기
3. 삭제 버튼 클릭 → 확인 후 삭제

**완료 조건**
- 새 프로젝트가 1클릭으로 생성되고 바로 작업 시작 가능
- 프로젝트 목록이 최근 수정 순으로 정렬됨

### 화면 2) Insight Workspace (3분할 메인)

#### 좌측 패널: 입력 & 컨텍스트

**요소**
- 레퍼런스 텍스트 입력창 (Textarea)
  - 플레이스홀더: "분석하고 싶은 영상의 자막이나 대본을 붙여넣으세요..."
  - 글자수 표시
- 내 메모 입력창 (Textarea)
  - 플레이스홀더: "내가 전달하고 싶은 내용을 자유롭게 작성하세요..."
  - 글자수 표시
- 경고 박스: "💡 레퍼런스는 구조 참고용입니다. 문장을 복제하지 않도록 AI가 조정합니다."

**유저 액션**
1. 레퍼런스 영상 자막/대본 붙여넣기
2. 내 경험/자료 메모 작성

**제약 조건**
- 레퍼런스 200자 미만 → 경고 (나중에 추가 가능)
- 내 메모 50자 미만 → 경고 (나중에 추가 가능)

**완료 조건**
- 입력 중 글자수가 실시간으로 보임
- 입력 내용이 자동 저장됨 (API 호출 시)

---

#### 중앙 패널: 설정 & 액션

**요소**
1. **톤앤매너 선택** (버튼 그룹)
   - 지식전달형: 차분하고 신뢰감 있는 어조
   - 엔터테이너형: 텐션 높고 유머러스한 어조
   - 담백정보형: 짧고 빠르게 핵심만 전달

2. **후킹 스타일 선택** (버튼 그룹)
   - 호기심: "이걸 모르면 손해"
   - 이득: "이렇게 하면 ~를 얻을 수 있다"
   - 공포/리스크: "이걸 안 하면 위험하다"

3. **목표 길이 선택** (버튼 그룹)
   - 3분 / 6분 / 10분

4. **액션 버튼**
   - "🔍 분석하기": 인사이트 카드 생성
   - "✨ 대본 생성하기": 2-Column 스크립트 생성

5. **LLM Provider 선택** (드롭다운, 우측 상단)
   - OpenAI / Anthropic

**유저 액션**
1. 톤앤매너 선택 (1개만 활성화)
2. 후킹 스타일 선택 (1개만 활성화)
3. 길이 선택 (1개만 활성화)
4. "분석하기" 클릭 → 우측 패널에 인사이트 카드 생성
5. "대본 생성하기" 클릭 → 우측 패널에 2-Column 스크립트 생성

**완료 조건**
- 입력 부족 시 버튼 비활성화
- 생성 중 버튼 텍스트 변경 ("분석 중..." / "생성 중...")
- 생성 완료 시 소요 시간 표시

---

#### 우측 패널: 결과 & Export

**초기 상태**
- "👈 좌측에서 입력 후 분석 또는 대본 생성을 시작하세요"

**1) 분석 결과 (인사이트 카드)**

**요소**
- **구조 분석** (4단계)
  - Hook: 후킹 단계 요약
  - Body: 본론 전개 요약
  - Turn: 전환/위기 단계
  - Conclusion: 결론/CTA
- **인사이트 카드** (5~10개)
  - 제목 (예: "3초 이내 숫자로 호기심 자극")
  - 설명 (예: "영상 시작 3초 안에 구체적 숫자를 제시...")
  - 배지 (추천 위치: hook/body/turn/conclusion)
  - 강도 (1~5, 점으로 표시)
  - 체크박스 (선택/해제)

**유저 액션**
1. 카드 클릭 또는 체크박스 선택 → 대본 생성 시 반영
2. 선택한 카드만 활용해 "대본 생성하기" 재실행

**완료 조건**
- 체크된 카드만 대본에 반영됨
- 카드 선택/해제가 즉시 반영됨

---

**2) 대본 결과 (2-Column 스크립트)**

**요소**
- **제목 후보** (5개)
  - 예: "1. 지금 바로 시작하는 AI 활용법"
- **썸네일 컨셉** (3개)
  - 구도 + 텍스트
  - 예: "중앙 인물 + "10분 안에 끝내기""
- **2-Column 스크립트** (표 형태)
  - 좌측 열: Visual (화면/자막/효과 가이드)
  - 우측 열: Audio (실제 내레이션)
  - 타이밍 힌트 (선택 사항)

**유저 액션**
1. 스크립트 행 클릭 → 리터칭 패널 활성화
2. 리터칭 버튼 클릭 ("더 짧게" / "더 자극적으로" / "유머 추가")
3. 수정 결과 확인 → "적용" 또는 "취소"
4. "📄 Markdown 다운로드" 클릭 → .md 파일 다운로드
5. "📋 클립보드 복사" 클릭 → 전체 대본 복사

**완료 조건**
- 리터칭이 선택한 행에만 적용됨
- Export 결과물이 2-Column 형식 유지
- 다운로드 파일명이 프로젝트 제목

---

## 3. 데이터 흐름 (비기술 용어)

### 분석하기 흐름
1. 사용자가 레퍼런스 + 내 메모 입력
2. "분석하기" 클릭
3. AI가 레퍼런스 구조 분석 → 인사이트 카드 생성
4. 우측 패널에 결과 표시

### 대본 생성하기 흐름
1. 톤앤매너/후킹/길이 설정
2. (선택) 인사이트 카드 선택
3. "대본 생성하기" 클릭
4. AI가 레퍼런스 구조 + 내 메모 결합 → 2-Column 대본 생성
5. 우측 패널에 제목/썸네일/스크립트 표시

### 리터칭 흐름
1. 스크립트 특정 행 클릭
2. 리터칭 스타일 선택
3. AI가 해당 문장만 수정
4. 수정 결과 확인 후 적용/취소

## 4. 확장 가능 지점 (PM이 추가할 수 있는 기능)

### A. 새 프리셋 추가
**현재**: 지식전달형 / 엔터테이너형 / 담백정보형

**추가 예시**
- 스토리텔링형: 서사 중심, 감정 호소
- B2B 전문형: 논리적, 데이터 중심

**추가 시 필요한 정보**
- 프리셋 이름
- 특징 (어조, 문장 길이, 강조점)
- 예시 문장

---

### B. 인사이트 카드 타입 확장
**현재**: 제목/설명/위치/강도

**추가 예시**
- 카드 타입: 패턴 / 비유 / 사례 / 전환 장치
- 적용 난이도: 쉬움 / 보통 / 어려움
- 참고 영상 링크

---

### C. 타겟 시청자 페르소나 입력
**추가할 필드**
- 타겟 시청자: "20대 직장인" / "중소기업 대표" 등
- 시청 목적: "문제 해결" / "정보 습득" / "엔터테인먼트"

**활용 방식**
- 대본 생성 시 타겟 맞춤 어휘/사례 사용

---

### D. 히스토리/버전 관리
**기능**
- 같은 프로젝트의 여러 대본 버전 저장
- 버전 비교 (A/B 테스트)
- 되돌리기

---

### E. 썸네일 이미지 생성 (고도화)
**현재**: 텍스트 컨셉만 제공

**추가**
- DALL-E/Midjourney 연동
- 실제 썸네일 이미지 생성

---

## 5. 제약 사항 및 정책

### MVP에서 의도적으로 제외한 기능
- 유튜브 URL 자동 자막 추출 (저작권/복잡도)
- 댓글/트렌드 자동 수집 (API 비용/복잡도)
- 이미지 생성 (비용/시간)
- 협업/공유 기능 (초기 불필요)
- 결제/구독 (검증 후 추가)

### 개인정보/저작권 정책
- 사용자 입력 텍스트는 생성 목적으로만 사용
- LLM 모델 학습에 사용하지 않음 (OpenAI/Anthropic 정책 준수)
- 레퍼런스 영상 문장 복제 지양 (프롬프트에 명시)

## 6. 성과 측정 (Dogfooding)

### 측정 항목
- **제작시간 단축**: 입력 시작 → 초안 완료까지 시간
- **재생성 횟수**: 만족할 때까지 몇 번 생성했는지
- **Export 사용률**: 실제로 다운로드/복사했는지
- **토큰 사용량**: 비용 추정

### 활용 방식
- 프로젝트별 메트릭 확인 (개발자 도구)
- 전체 평균 메트릭으로 개선 포인트 발견

## 7. 용어 사전

| 용어 | 의미 |
|------|------|
| 레퍼런스 텍스트 | 벤치마킹하고 싶은 영상의 자막/대본 |
| 내 메모 | 내 경험/관점/자료 |
| 인사이트 카드 | 레퍼런스에서 추출한 효과적인 패턴 |

| 톤앤매너 | 영상의 전반적인 어조/분위기 |
| 후킹 스타일 | 영상 첫 10초의 시선 끌기 방식 |
| 리터칭 | 특정 문장을 스타일에 맞게 재작성 |
| LLM Provider | AI 생성 엔진 (OpenAI/Anthropic) |

## 8. FAQ (PM용)

**Q. 새 프리셋을 추가하려면?**
A. 개발자에게 "프리셋 이름 + 특징 + 예시 문장"을 전달하세요.

**Q. 인사이트 카드 개수를 조정할 수 있나요?**
A. 네, 프롬프트 수정으로 가능합니다. (최소 5개, 최대 10개 → 원하는 범위로 변경)

**Q. 대본 길이가 목표와 안 맞을 때는?**
A. "목표 길이" 설정이 가이드일 뿐, 정확히 맞추지는 않습니다. 프롬프트 개선 필요.

**Q. 여러 언어 지원 가능한가요?**
A. 현재는 한국어 중심. 프롬프트 다국어화 필요.

**Q. 비용은 얼마나 드나요?**
A. 분석 1회 약 $0.02~0.05, 대본 1회 약 $0.05~0.10 (모델/길이에 따라 변동)
